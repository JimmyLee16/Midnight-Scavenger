## Hướng dẫn sử dụng nhanh

** Chuẩn bị **

- Cardano address source https://github.com/IntersectMBO/cardano-addresses/releases
- Cardano cli source https://github.com/IntersectMBO/cardano-cli/releases
- Cardano-signer.exe source (https://github.com/gitmachtl/cardano-signer/releases/tag/v1.32.0)
- Download các file ps1

** Sử dụng ** 
- Đặt chung tất cả các công cụ và file 1 thư mục Chạy file .bat
- Lưu ý: 
  + Các file .ps1 cần save as UTF-8 with BOM
  + thư mục Cardano-cli nên đổi tên thành Cardano-cli-win64 (hoặc để tên tùy ý nhưng cần đổi lại tên đường dẫn trong các file script)
